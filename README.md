# -statistiksel-Tutum-Analizi-Hacettepe-rencilierinin-Ya-l-lara-Y-nelik-Alg-lar-
Hacettepe Üniversitesi öğrencileri üzerinde yürütülen, yaşlı bireylere yönelik tutum, algı ve davranış kalıplarının kategorik veri analizi yöntemleriyle incelendiği 2025-2026 dönem projesi. Çalışmada odds oranları, uyum analizi (biplot), lojistik regresyon ve log-lineer modelleme teknikleri kullanılmıştır.
